# 🌲🏆 PokeBug — 探索模式 & 甲虫杯设计 v1.0

> 基于现有 MVP 的扩展玩法设计
> 设计日期：2026-06-19
> 整合版本：v1.0

---

## 1. 设计定位

| 系统 | 定位 | 触发方式 | 对手 | 核心乐趣 |
|---|---|---|---|---|
| **探索模式** | 日常冒险 | 主动菜单选择 | NPC（4档） | 收集资源、冒险、放生传代 |
| **甲虫杯** | 定期大考 | 2小时自动触发 | NPC（杯赛专属分布） | 赛季荣誉、淘汰赛刺激 |
| **ESP-NOW 对战** | 社交竞技 | 手动选择 | 真人玩家 | 联机对战 |

- **探索是日常**：成虫离开培养缸，进入野外随机事件——获取食物、遭遇 NPC、选择放生。
- **甲虫杯是赛季**：每 2 小时自动触发一次淘汰赛，给养成一个明确目标——"下次杯赛我能走多远？"
- 两者共用 **NPC 4 档属性体系**（见 §5），但杯赛 NPC 分布更偏强。
- 探索获取的资源（树汁/SPI）用来喂养甲虫 → 属性提升后在杯赛检验 → 杯赛奖励反哺养成 → 形成 **探索→养成→杯赛→更强→再探索** 的闭环。

## 2. 入口与条件

### 2.1 探索模式入口

菜单新增 🌲 探索图标：

```
┌─────────────────────────────────────────────────────┐
│         (培养缸半暗背景)                              │
│                                                       │
│           🍎  🪵  ⚔️  🌲  ❓                          │
│          喂食 腐木 对战 探索 信息                       │
│                       ▲                               │
└─────────────────────────────────────────────────────┘
```

### 2.2 进入条件

| 条件 | 探索模式 | 甲虫杯 |
|---|---|---|
| 阶段 = 成虫 | ✅ | ✅ |
| 饥饿度 ≥ 30 | ✅ | ✅ |
| 未在对战中 | ✅ | ✅ |
| 距上次杯赛 ≥ 2 小时 | — | ✅（全局冷却） |

### 2.3 甲虫杯触发流程

```
杯赛时间到 → LED 闪烁（金色）→ 屏幕弹出通知
                                         │
                              玩家短按A → 进入杯赛
                              玩家短按B → 放弃（本次跳过）
                              30秒无操作 → 自动放弃
```

**通知画面**：

```
┌─────────────────────────────────────────────────────┐
│                                                       │
│                    🏆                                │
│              甲虫杯 即将开始！                         │
│                                                       │
│          第 12 届 甲虫杯                              │
│          你的最佳战绩：🏅 冠军                         │
│                                                       │
│        [短按A参赛]    [短按B放弃]                      │
└─────────────────────────────────────────────────────┘
```

> 2 小时一届：一只成虫存活几小时到几天，一生能参加 3-10 次杯赛。玩家能感受到"这只虫越养越强，杯赛名次越来越好"的成长曲线。

### 2.4 杯赛触发时机

```
设备启动 → 检查 last_cup_time
              │
    距上次 < 2小时 → 设置定时器，到点触发
    距上次 ≥ 2小时 → 立即触发（补偿）
    首次启动     → 2小时后触发
```

- 定时器在设备 awake 时检查
- 如果在 deep sleep 期间错过，醒来后立即触发
- 不会因为休眠而错过杯赛

┌─────────────────────────────────────────────────────┐
│ ☀️                                                  │

### 3.2 移动方式

- **倾斜控制**：左倾→甲虫向左走，右倾→向右走
- 甲虫走到边缘自动回头
- 每 5-10 秒触发一次随机事件

| 消耗 | 数值 | 说明 |
|---|---|---|
| 饥饿度 | -20/次 | 进探索时扣，饿着回来就得赶紧喂 |
| 探索时长 | 30-60 秒 | 随机事件间隔 5-10 秒 |
| 探索上限 | 无限制 | 只要饥饿度够，随时能出门 |
| 探索冷却 | 无 | 出来就能立刻再进 |

## 4. 探索事件系统

### 4.1 事件概率表

| 事件 | 概率 | 效果 |
|---|---|---|
| 🌳 发现树汁 | 28% | 树汁 +1~3 |
| ⚔️ 遭遇对手 | 25% | 触发 NPC 对战（见 §5） |
| 🪵 发现腐木 | 10% | 腐木 +1（未满时） |
| 🍂 发现食物源 | 20% | 树汁 +1~2 |
| ✨ 稀有事件 | 7% | 高回报特殊事件（见 §4.3） |
| 💨 无事发生 | 10% | 无 |

### 4.2 资源发现事件

#### 🌳 发现树汁 / 🍂 发现食物源

```
┌─────────────────────────────────────────────────────┐
│                                                       │
│              🌳                                       │
│           发现了树汁！                                 │
│           +2 🍎                                      │
│                                                       │
│              [短按A继续]                               │
└─────────────────────────────────────────────────────┘
```

- 树汁 1-3 份，食物源 1-2 份
- 超过背包上限的部分丢弃

#### 🪵 发现腐木

- 仅当腐木未放置且背包未满时触发
- 已有腐木时此事件替换为"发现树汁"

### 4.3 稀有事件

低概率高回报的特殊遭遇。子概率 = 在"稀有事件"（7%）中的占比。

| 稀有事件 | 子概率 | 效果 | 画面描述 |
|---|---|---|---|
| 🌟 金色树汁 | 35% | 树汁 +5，满上背包 | 阳光透过树叶照在金色树脂上 |
| 💎 树脂结晶 | 20% | 树汁 +3，下次喂食 END 额外 +0.5 | 树汁凝结成琥珀色晶体 |
| 🦋 蝴蝶引路 | 15% | 立即触发一次额外普通事件 | 蝴蝶在甲虫前方飞舞 |
| 🌧️ 雨后菌丛 | 15% | 树汁 +3，腐木 +1 | 雨后草地上冒出一片菌菇 |
| 🍯 蜜露喷泉 | 10% | 树汁 +6（溢出丢弃） | 树干裂缝渗出金色蜜露 |
| 👻 幻影甲虫 | 5% | 树汁 +5，腐木 +1，SPI +0.5（不战斗直接送） | 半透明巨大甲虫虚影掠过 |

> 幻影甲虫总概率仅 0.35%，比传说 NPC 还稀有。是森林的祝福，不需战斗，画面短暂显示半透明巨大甲虫剪影后消散。

## 5. NPC 对战体系（探索 & 杯赛共用）

### 5.1 4 档 NPC 体系

| 档位 | 名字 | 属性定位 | 遭遇概率 | 属性范围 | 对话风格 |
|---|---|---|---|---|---|
| 🟢 新手 | 虫虫小朋友、邻居小美、路过的阿柴 | 弱于玩家 | 40% | 玩家属性 × 0.5~0.8 | 天真、紧张、崇拜 |
| 🟡 普通 | 虫太郎、甲虫阿姨、公园老陈、山田大叔 | 与玩家相当 | 35% | 玩家属性 × 0.8~1.2 | 日常、认真、友好 |
| 🟠 老手 | 甲斗王阿猛、深山猎手、独角尊者 | 强于玩家 | 20% | 玩家属性 × 1.2~1.6 | 自信、挑衅、前辈口吻 |
| 🔴 传说 | 虫神龙之介、黑角仙人、森之守护者 | 碾压级 | 5% | 玩家属性 × 1.8~2.5 | 神秘、威严、诗意 |

> 探索中传说档总遭遇率 = 25%（遭遇对手） × 5% = **1.25%**，极其稀有。

### 5.2 NPC 属性生成

```c
int tier = random_tier();  // 加权随机
float mult_min, mult_max;
switch (tier) {
    case TIER_ROOKIE:  mult_min=0.5, mult_max=0.8;  break;
    case TIER_NORMAL:  mult_min=0.8, mult_max=1.2;  break;
    case TIER_VETERAN: mult_min=1.2, mult_max=1.6;  break;
    case TIER_LEGEND:  mult_min=1.8, mult_max=2.5;  break;
}
// 每个属性独立随机
npc_attr = clamp((int)(player_attr * random_float(mult_min, mult_max)), 1, 10);
npc_mot = 50 + random(-15, 15);
```

## 6. 甲虫杯赛制

### 6.1 淘汰赛结构

```
          8强            4强           决赛
      ┌── 🪲A ──┐
      │         ├── 🪲A ──┐
      ├── 🪲B ──┘         │
      │                   ├── 🪲A ──┐
      ├── 🪲C ──┐         │         │
      │         ├── 🪲D ──┘         │
      ├── 🪲D ──┘                   │
      │                             ├── 🏆 冠军!
      ├── 🪲E ──┐                   │
      │         ├── 🪲F ──┐         │
      ├── 🪲F ──┘         │         │
      │                   ├── 🪲H ──┘
      ├── 🪲G ──┐         │
      │         ├── 🪲H ──┘
      ├── 🪲H ──┘

   第1轮（8进4）    第2轮（半决赛）  第3轮（决赛）
```

- 玩家甲虫永远在 A 位置
- 其余 7 个对手为 NPC
- 每轮对战玩家都可以看，自己的比赛可以按A加油
- 输一轮即淘汰，比赛结束

### 6.2 杯赛 NPC 档位分布

杯赛 NPC 档位分布比探索更偏强——来参赛的都是有备而来的。

| 档位 | 杯赛概率 | 探索概率（对比） | 说明 |
|---|---|---|---|
| 🟢 新手 | 10% | 40% | 杯赛里新人是来凑热闹的 |
| 🟡 普通 | 30% | 35% | 参赛主力 |
| 🟠 老手 | 40% | 20% | 杯赛常见对手 |
| 🔴 传说 | 20% | 5% | 杯赛传说比例大幅提高 |

> 运气好第一轮抽到新手，运气不好第一轮就撞传说。但赢了传说的成就感也是最高的。

## 8. 放生机制 🌿

探索模式的核心情感设计。不想继续养这只甲虫了？不想等它饿死？在野外放生它，它会留下一颗卵——基因延续，世代传承。

### 8.1 触发方式

探索中 **长按A** → 弹出放生确认

```
┌─────────────────────────────────────────────────────┐
│                                                       │
│          🪲  →  🌿                                   │
│                                                       │
│       放生这只甲虫吗？                                 │
│       它会留下一颗卵...                                │
│                                                       │
│       [短按A确认]    [短按B取消]                       │
└─────────────────────────────────────────────────────┘
```

### 8.2 放生后流程

1. 播放放生动画：甲虫走向草丛深处，消失
2. 屏幕显示卵落下动画
3. 回到培养缸，画面变为一颗新卵

### 8.3 基因继承

放生产生的卵**继承父母基因，有小幅变异**：

```c
// 基因继承规则
child_gene = parent_gene;  // 基础继承
// 5% 概率每个基因发生变异
for each gene:
    if (random(100) < 5):
        gene = clamp(gene + random(-2, 2), 0, 15);
```

| 继承 | 说明 |
|---|---|
| 基因 | 继承 + 5% 变异 |
| 世代数 | generation +1 |
| 战绩 | 清零（新生命，新开始） |
| 其他属性 | 全部重置为初始值 |

### 8.4 战略意义

| 场景 | 策略 |
|---|---|
| 甲虫属性差 | 放生碰运气——基因变异可能更好 |
| 甲虫属性好 | 放生保留好基因，下一代可能更强 |
| 养腻了 | 不用等饿死，主动换新虫 |
| 想刷基因 | 反复放生→变异筛选（耗时但可行） |
