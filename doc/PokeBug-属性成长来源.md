# PokeBug 属性成长来源

记录时间：2026-06-23

本文记录当前源码中 `SIZ / STR / END / SPD / SPI` 的所有直接成长、负增长和成长倍率来源，用于后续养成与基因培育平衡调整。

## 1. 基础规则

所有属性最终都会经过 `Bug::clampAttributes()` 限制：

- 属性最低为 `1.0`。
- 属性最高不能超过对应基因上限。
- 因此任何成长来源都不会突破当前基因潜力上限。

基因与属性对应关系：

| 属性 | 对应基因 | 显性影响 | 显性+隐性影响 |
|---|---|---|---|
| SIZ | VIG | SIZ 成长倍率 | SIZ 上限 |
| STR | ATK | STR 成长倍率 | STR 上限 |
| END | END | END 成长倍率 | END 上限 |
| SPD | APP | SPD 成长倍率 | SPD 上限，同时影响外观调色板 |
| SPI | MNT | SPI 成长倍率 | SPI 上限 |

成长倍率公式：

```cpp
growthMult = 0.8 + dominantGene * 0.1
```

显性值范围为 `0-15`，因此成长倍率范围为 `0.8x - 2.3x`。

属性上限公式：

```cpp
cap = 6 + (dominant + recessive) / 2
```

显性和隐性值范围均为 `0-15`，因此单项属性上限范围为 `6-21`。

## 2. 食物成长

食物成长公式：

```cpp
delta = foodBase * stageAbsorb * temperamentMult * environmentMult * geneGrowthMult * penalty
```

其中：

- `foodBase` 来自食物属性向量。
- `stageAbsorb` 来自当前生命周期阶段。
- `temperamentMult` 来自卵期气质。
- `environmentMult` 来自食物盘等级、食物倾向和成虫腐木倾向。
- `geneGrowthMult` 来自对应显性基因。
- `penalty` 当前主要用于 Jelly 间隔惩罚。

### 2.1 食物基础值

| 食物 | SIZ | STR | END | SPD | SPI | 每口 HUN |
|---|---:|---:|---:|---:|---:|---:|
| Drop | 0.30 | 0.05 | 0.00 | 0.00 | 0.00 | 15 |
| Cube | 0.05 | 0.30 | 0.00 | 0.00 | 0.00 | 13 |
| Slice | 0.15 | 0.15 | 0.00 | 0.15 | 0.00 | 11 |
| Citrus | 0.10 | 0.08 | 0.00 | 0.05 | 0.15 | 12 |
| Jelly | 0.20 | 0.25 | 0.00 | 0.20 | 0.00 | 8 |
| Berry | 0.15 | 0.15 | 0.15 | 0.10 | 0.10 | 9 |

每份食物分成 5 口吃完。Drop 是基础食物，恢复 HUN 最高；高等级或属性收益更强的食物恢复 HUN 较少，用来平衡“涨属性”和“吃得饱”的收益。

### 2.2 阶段吸收率

| 阶段 | SIZ | STR | END | SPD | SPI |
|---|---:|---:|---:|---:|---:|
| 幼虫 | 1.00 | 1.00 | 0.00 | 0.20 | 0.10 |
| 青年 | 0.50 | 0.50 | 0.50 | 1.00 | 0.50 |
| 成虫 | 0.15 | 0.15 | 0.15 | 0.15 | 0.15 |

设计含义：

- 幼虫期主要成长 `SIZ / STR`。
- 青年期适合补 `SPD / END / SPI`。
- 成虫期可以补全部属性，但效率较低，适合作为后期微调。

### 2.3 气质倍率

| 气质 | 正向影响 | 负向影响 |
|---|---|---|
| SWIFT | SPD 1.10x | SIZ 0.90x |
| RESILIENT | END 1.10x | SPI 0.90x |
| GIANT | SIZ 1.10x | SPD 0.90x |
| BRUTE | STR 1.10x | END 0.90x |
| BALANCED | 无修正 | 无修正 |
| SPIRIT | SPI 1.10x | STR 0.90x |

### 2.4 环境倍率

食物盘等级：

- Lv.2：全部食物成长 `1.03x`。
- Lv.3：全部食物成长 `1.06x`。

食物倾向：

- 当前食物与属性倾向匹配时，该属性额外 `1.03x`。

成虫腐木倾向：

- 仅成虫期、且已放置腐木时生效。
- 腐木风格与属性匹配时，该属性额外 `1.03x`。

## 3. 阶段固定成长

阶段推进会直接增加属性：

| 阶段变化 | 属性变化 |
|---|---|
| 幼虫 -> 蛹 | STR +1.0，SIZ +0.5 |
| 蛹 -> 青年 | SPI += 3.0 - pupaShakes |

注意：

- 蛹期摇晃次数 `pupaShakes` 会降低蛹化结束时的 SPI 奖励。
- 该奖励仍然会被 SPI 上限 clamp。

## 4. 蛹期静养

蛹期会安静成长 SPI：

```cpp
SPI +1.0
```

当前蛹期为 20 分钟，`PUPA_SPI_GROWTH_MS` 也跟随蛹期总时长，因此整个蛹期稳定总计 `SPI +1.0`。

如果幼虫期喂过 Berry，间隔会缩短 10%，但因为蛹期总时长不变，当前实现仍主要表现为稳定触发一次。

## 5. 成虫行为成长

### 5.1 摇晃训练

成虫期摇晃设备：

```cpp
STR +0.1
MOT +20
```

限制：

- 每 3 分钟最多触发一次。
- STR 仍受 STR 上限限制。

### 5.2 腐木休息

成虫期满足以下条件时，腐木休息可增加 END：

- 当前为成虫。
- 已放置腐木。
- 饥饿度不低于 50。
- 甲虫实际趴在腐木旁休息。

每累计 4 分钟休息：

```cpp
END += 0.2 * endGrowthMult * temperamentEndMult
```

其中：

- `endGrowthMult` 来自 END 显性基因。
- RESILIENT 气质为 `1.10x`。
- BRUTE 气质为 `0.90x`。

腐木休息是当前 END 最稳定的后期成长来源。

## 6. 探索成长

探索普通事件主要提供食物或腐木。直接属性成长只出现在部分 rare 子事件中。

rare 事件概率由地点和时段决定，大致范围为 `5%-30%`；触发 rare 后再从 6 个子事件中随机。

| 地点 | 子事件效果 |
|---|---|
| Park | SPI +0.1 |
| Back Hill | END +0.1 |
| Riverside | SPI +0.2，或 END +0.2 |
| Old Woods | END +0.3，或 STR +0.2，或 SPI +0.3 |

探索属性奖励调用 `addTrainingBonus()`，会直接增加属性并 clamp 到上限。

## 7. 战斗成长

战斗只直接增加 SPI。

### 7.1 本地 NPC 战

本地 NPC 战胜利后，根据 NPC tier 随机获得 SPI：

| NPC Tier | 触发概率 | SPI 奖励 |
|---|---:|---:|
| Rookie | 30% | +0.15 |
| Normal | 45% | +0.15 ~ +0.30 |
| Veteran | 60% | +0.30 ~ +0.45 |
| Legend | 80% | +0.45 ~ +0.60 |

失败时不获得 SPI 奖励。

### 7.2 非 NPC 战默认奖励

如果不是本地 NPC 战，当前默认奖励为：

| 结果 | SPI 奖励 |
|---|---:|
| 胜利 | +0.5 |
| 失败 | +0.2 |

## 8. 负增长来源

当前只有 SPI 存在明确负增长。

| 行为 | 阶段 | 效果 |
|---|---|---|
| 摇晃 | 幼虫 | SPI -0.2 |
| 摇晃 | 蛹 | SPI -1.0，并增加 `pupaShakes` |

蛹期摇晃会产生双重影响：

- 立即 `SPI -1.0`。
- 降低蛹 -> 青年时的 `SPI += 3.0 - pupaShakes` 奖励。

## 9. 各属性来源汇总

### SIZ

直接来源：

- Drop、Cube、Slice、Citrus、Jelly、Berry 食物成长。
- 幼虫 -> 蛹固定 `SIZ +0.5`。

间接影响：

- VIG 显性基因影响 SIZ 成长倍率。
- VIG 显性+隐性影响 SIZ 上限。
- GIANT 气质提升 SIZ 成长。
- SWIFT 气质降低 SIZ 成长。
- Stack 腐木在成虫期提供 SIZ 环境倍率。

### STR

直接来源：

- Drop、Cube、Slice、Citrus、Jelly、Berry 食物成长。
- 幼虫 -> 蛹固定 `STR +1.0`。
- 成虫摇晃训练 `STR +0.1`。
- Old Woods rare 子事件 `STR +0.2`。

间接影响：

- ATK 显性基因影响 STR 成长倍率。
- ATK 显性+隐性影响 STR 上限。
- BRUTE 气质提升 STR 成长。
- SPIRIT 气质降低 STR 成长。
- Twig 腐木在成虫期提供 STR 环境倍率。

### END

直接来源：

- Berry 食物成长。
- 成虫腐木休息。
- Back Hill rare 子事件 `END +0.1`。
- Riverside rare 子事件 `END +0.2`。
- Old Woods rare 子事件 `END +0.3`。

间接影响：

- END 显性基因影响 END 成长倍率。
- END 显性+隐性影响 END 上限。
- RESILIENT 气质提升 END 成长。
- BRUTE 气质降低 END 成长。
- Hollow 腐木在成虫期提供 END 环境倍率。

### SPD

直接来源：

- Slice、Citrus、Jelly、Berry 食物成长。

间接影响：

- APP 显性基因影响 SPD 成长倍率。
- APP 显性+隐性影响 SPD 上限。
- APP 显性基因同时影响外观调色板。
- SWIFT 气质提升 SPD 成长。
- GIANT 气质降低 SPD 成长。
- Pale 腐木在成虫期提供 SPD 环境倍率。

### SPI

直接来源：

- Citrus、Berry 食物成长。
- 蛹期静养 `SPI +1.0`。
- 蛹 -> 青年 `SPI += 3.0 - pupaShakes`。
- 探索 rare 子事件：`SPI +0.1 / +0.2 / +0.3`。
- 本地 NPC 战胜利随机 SPI 奖励。
- 非 NPC 战默认 SPI 奖励。

负增长：

- 幼虫摇晃 `SPI -0.2`。
- 蛹期摇晃 `SPI -1.0`，并降低蛹化结束奖励。

间接影响：

- MNT 显性基因影响 SPI 成长倍率。
- MNT 显性+隐性影响 SPI 上限。
- SPIRIT 气质提升 SPI 成长。
- RESILIENT 气质降低 SPI 成长。
- Mossy 腐木在成虫期提供 SPI 环境倍率。

## 10. 当前注意点

### 10.1 Citrus 的 SPI 上限奖励可能未实际生效

幼虫期累计喂 3 次 Citrus 会设置：

```cpp
spiCapBonusTenths = 3;
```

设计意图像是 `SPI 上限 +0.3`。

但当前 `spiCap()` 返回 `uint8_t`，小数部分会被截断，因此 `+0.3` 基本不会改变最终上限。

如果希望该机制生效，需要改为：

- 将 SPI cap 改成浮点参与 clamp；或
- 将 bonus 改为整数上限奖励；或
- 累计到 `+1.0` 时才提升 cap。

### 10.2 成虫食物补强不宜作为主要练满路径

成虫吸收率当前为 `0.15`，比旧值更友好，但仍明显低于幼虫和青年期。

设计上更适合：

- 幼虫期决定体型和力量基础。
- 青年期补速度、耐力和精神。
- 成虫期做微调、训练、探索、战斗与遗传前冲刺。

### 10.3 遗传逻辑使用已达属性阶段分

当前放生继承使用 `当前属性 / 当前上限` 的阶段分作为遗传压力：

- `<50%`
- `50%-69%`
- `70%-84%`
- `85%-94%`
- `>=95%`

这意味着玩家不需要把每项属性练满才有遗传收益；达到 70%-85% 已经能形成比较明显的下一代加权。
